# Gerenciamento de Mídia Digital (POO)
Este projeto foi desenvolvido como atividade acadêmica para praticar os conceitos de Programação Orientada a Objetos.
O sistema simula itens de mídia digital, como Filmes e Músicas, permitindo reproduzir e avaliar cada um deles.

--- Foram utilizados conceitos como classe abstrata, interface, herança, encapsulamento e polimorfismo para estruturar o código.

--- O arquivo TestePOOCompleto demonstra o funcionamento criando objetos, chamando seus métodos e testando os comportamentos de cada classe.
